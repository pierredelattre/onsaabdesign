# onsaabdesign — PRD

## Vision produit

Une archive de référence sur Saab comme objet de design industriel. Pensée pour durer, pas pour être virale. Chaque fiche est un travail d'édition, pas un article de blog.

## Objectifs

1. Constituer un vault de 50+ fiches qualitatives sur la marque, ses ingénieurs, ses modèles, son ergonomie et ses connexions
2. Être la ressource de référence pour quiconque cherche à comprendre Saab sous l'angle design
3. Documenter avant que les sources primaires disparaissent davantage

## North Star Metric

**Profondeur du vault** : nombre de fiches complètes (texte + sources + connexions). Objectif initial : 20 fiches de qualité avant toute publication publique.

## MoSCoW

### Must have
- Template de fiche standardisé (frontmatter YAML + structure corps)
- Vault avec au minimum : 3 fiches philosophie, 3 fiches ergonomie, 3 fiches people, 5 fiches modèles, 3 fiches connexions
- Disclaimer visible partout
- Système de cross-références entre fiches (wikilinks)

### Should have
- Index par catégorie et par tag
- Timeline Saab 1947-2011
- Page "À propos / Manifeste"
- Publication lisible en ligne (site statique minimaliste)

### Could have
- Newsletter pour les analyses approfondies
- Série comparatives Saab vs Braun
- Galerie visuelle sourcée

### Won't have (pour l'instant)
- Comptes utilisateurs
- Commentaires / forum
- E-commerce ou merch
- Contenu non sourcé
- Reproduction de visuels officiels Saab protégés

## Anti-scope

Pas un forum. Pas un site de pièces détachées. Pas une plateforme communautaire. Pas de contenu généré par les visiteurs. Pas de représentation officielle de Saab AB ou NEVS.
