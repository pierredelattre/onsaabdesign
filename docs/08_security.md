# onsaabdesign — Sécurité & droits

## Risques principaux

Ce projet n'a pas de backend, pas de comptes utilisateurs, pas de données personnelles. Les risques de sécurité informatique sont minimes.

Les risques réels sont éditoriaux et juridiques.

## Droits de contenu

### Textes
Tout le contenu textuel est original. Pas de copié-collé de sources officielles Saab. Citations courtes acceptées avec référence explicite.

### Visuels
Règle stricte :
- **Photos personnelles** : OK
- **Photos sous licence Creative Commons** : OK avec crédit
- **Photos presse avec autorisation** : OK
- **Visuels officiels Saab (site, brochures, chartes)** : NON sans autorisation écrite
- **Captures d'écran de livres/magazines** : zone grise — usage éditorial minimal, bien sourcé

Chaque image dans le vault doit avoir une ligne `source:` dans son frontmatter ou sa légende.

## Disclaimer obligatoire

À afficher dans :
- Le README
- Le CLAUDE.md
- Chaque fiche publiée en ligne
- La page "À propos" du site
- La bio des réseaux sociaux

Texte : *"Independent editorial project. Not affiliated with Saab AB or NEVS."*

## Marque & trademark

- "Saab" utilisé de façon nominative pour désigner le sujet — usage éditorial défendable
- Pas de reproduction du logo griffin ou du wordmark Saab en header, favicon, ou avatar
- Pas de merch utilisant la marque Saab sans autorisation écrite

## Phase publication

Site statique, pas de formulaire, pas de base de données, pas de cookies tiers. Si newsletter : utiliser une plateforme RGPD-compliant (Substack, Buttondown).
