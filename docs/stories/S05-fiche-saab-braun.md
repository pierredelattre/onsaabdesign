# S05 — Fiche connexion Saab / Braun

## Objectif
Première fiche "connexions" — le parallèle le plus fort de l'archive.

## Contexte
Saab et Braun partagent une philosophie proche : primauté de la fonction, refus de l'ornement, cohérence systématique entre les produits d'une même marque, dirigeants avec un point de vue sur le design. Braun sous Dieter Rams (1961-1995), Saab sous ses ingénieurs fondateurs (années 50-80). Les deux marques ont été absorbées par des groupes (Gillette/P&G pour Braun, GM pour Saab) qui ont progressivement dilué leur identité.
