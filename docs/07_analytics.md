# onsaabdesign — Analytics

## Phase second cerveau (actuelle)

Pas d'analytics. Le vault est local et privé.

Métriques de suivi manuel :
- Nombre de fiches complètes (statut "published")
- Couverture par catégorie (vault/INDEX.md)
- Nouvelles sources trouvées

## Phase publication (future)

Analytics minimaliste, sans cookies tiers. Recommandation : **Plausible Analytics** (privacy-first, RGPD-compliant, léger).

### Métriques utiles

| Métrique | Pourquoi |
|----------|----------|
| Pages les plus lues | Comprendre quels sujets intéressent |
| Sources de trafic | Valider les canaux de distribution |
| Temps passé par fiche | Indicateur de profondeur / qualité |
| Fiches les plus liées (backlinks) | Identifier les fiches "hub" |

### Ce qu'on ne suit pas

- Conversions (pas de conversion à mesurer)
- Funnel (pas de funnel)
- Données personnelles utilisateurs

## North Star Metric (rappel)

Nombre de fiches complètes dans le vault. Tout le reste est secondaire.
